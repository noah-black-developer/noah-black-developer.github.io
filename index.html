This page was designed as an experiment with building a website one piece of tech at a time, going from HTML to CSS to JavaScript. While modern web frameworks are powerful and brilliant at what they do, I was interested in getting a better understanding of the inner workings of a webpage and better inform my usage of those frameworks in the future.

I figure, well, since it was a bit of a deep-dive into the simpler days of the web, it only felt right to style the whole page as such. Which got us here - to a (hopefully) faithful recreation of the Apple II terminal screen!

Credit to some of the awesome folks on the internet who's references made this site possible:
  • Pat Orjk's Text-to-ASCII Generator, which helped create the styled text you see on this page.
  • Edwin on dev.to, for a great summary of how they went about creating a CSS-styled terminal of their own.
This whole page is open source, so feel free to view the project on Github if you're curious to see a bit of the inner workings! There might even be a few things I've kept secret, if you are wise enough to spot them...
